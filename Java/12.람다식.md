## 12.1 람다식이란?

- 함수(메서드)를 간단한 식으로 표현하는 방법 ⇒ 익명함수를 생성하기 위한 식

람다식은 `(매개변수) → {실행코드}` 형태로 작성

### 장점

1. 코드 간결
2. 컬렉션 요소를 필터링 하거나 매핑해서 원하는 결과를 쉽게 집계 가능

## 람다식 작성하기

1. 메서드의 이름과 반환타입을 제거


```java
int max(int v1, int v2) {
		return v1 > v2 ? v1 : v2;
}
```

2. `→` 를 중괄호 블록 { } 앞에 추가 

```java
~~int max~~(int v1, int v2) **->** {
		return v1 > v2 ? v1 : v2;
}
```

3. 반환값이 있는 경우에는 식이나 값만 적고 return문도 생략 가능 (세미콜론도 제거)

```java
(int v1, int v2) -> {
		return v1 > v2 ? v1 : v2;
}
```

```java
(int v1, int v2) **->** v1 > v2 ? v1 : v2
```
